# Research Profile Streamlit App

## Description
This is a simple Streamlit web application that presents a personal research profile.  
It allows users to view profile information, upload proposal files, explore academic and volunteering activities, and access contact details.

## Features
- Navigation sidebar with multiple sections
- Personal profile display
- CSV file upload and preview
- Academic and volunteering activity tables
- Contact information display

## Pages
- **My Profile**: Displays author details, study level, institution, and an image
- **Proposal**: Upload and view CSV proposal files
- **Activity**: View mentoring and volunteering activities
- **Contact**: Shows email address and a closing message

## Technologies Used
- Python
- Streamlit
- Pandas
